3  Technical SEO
Core Web Vitals
  • LCP < 2.5 s — optimize hero images (WebP, lazy-load below fold)
  • FID < 100 ms — defer non-critical JS, minimize main-thread work
  • CLS < 0.1 — set explicit width/height on all media elements
Crawlability & Indexing
  • XML sitemap at /sitemap.xml — auto-update on publish
  • robots.txt — allow all product pages, block /admin, /staging
  • Canonical tags on every page to prevent duplicate content
  • Self-referencing canonicals on paginated catalog pages
Structured Data
  • Organization schema — name, logo, contact, social profiles
  • Product schema on each product page — name, description, image, SKU, brand
  • LocalBusiness schema — address, geo-coordinates, hours
  • FAQPage schema on pillar pages for rich snippets
Performance & Security
  • HTTPS site-wide with HSTS header enabled
  • Mobile-first responsive design — test at 360 px breakpoint
  • Image compression pipeline: max 100 KB per image, WebP + fallback
  • Enable Brotli compression; set browser cache to 1 year for static assets
